# Student Management System (Python)

A simple Python-based console application to manage student records.

## Features
- Add student details
- View all students
- Search student by roll number
- Delete student records
- Menu-driven program

## Technologies Used
- Python

## How to Run
1. Open the project folder
2. Run the file:
   python student.py
